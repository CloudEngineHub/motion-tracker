# 匹克球视频测试指南

为了测试匹克球动作分析程序 (demos/pickleball_demo.py)，我们需要**从球员背后拍摄（背视角 / Back View）**的比赛视频。这类视角的视频可以最大程度且无遮挡地捕捉到球员的四肢和躯干动作。

## 如何获取此类视频？

获取这类视频主要有以下三种途径：

### 1. YouTube 搜索
YouTube 上有大量的匹克球比赛视频，这是最简单直接的途径。
*   **推荐搜索词**: 
    *   `pickleball unedited match back view` (匹克球 无剪辑 比赛 背视角)
    *   `pickleball raw court level` (匹克球 原始视频 场地平视角)
    *   `pickleball rec play back angle` (匹克球 休闲打法 背视角)
    *   `pickleball POV` (匹克球 第一人称/主视角 - 通常摄像机架在场地后方)
*   **挑选技巧**: 尽量选择摄像机**固定在场地后方围网上**或**从裁判椅后方拍摄**的视频。这类视频通常没有频繁的镜头切换，非常适合程序进行连续的动作追踪。可以使用如 `yt-dlp` 等工具将视频下载到本地进行测试。

### 2. 素材网站 (Stock Video Sites)
如果您需要更高清、甚至 4K 且没有转码压缩的高质量 "Raw" 视频，可以去商用素材网站查找。
*   **推荐网站**: Shutterstock, Freepik, Getty Images.
*   **推荐搜索词**: `pickleball court back view`, `pickleball match unedited`.
*   **注意**: 这些网站的视频通常需要付费购买，但有时会提供带水印的低分辨率预览版，可以先下载预览版测试算法是否能正常识别。

### 3. 自己拍摄 (最推荐)
如果您自己打匹克球，最理想的方式是自己拍摄一段素材。
*   **拍摄建议**: 
    1.  找个三脚架，或者使用能够固定在铁丝网上的手机支架（如 FenceMount）。
    2.  将相机/手机固定在**底线正后方**，高度约在 1.5 - 2 米（略高于球员头部或平视水平）。
    3.  确保画面能完整覆盖您这半场的球员全身（至少要能清楚看到脚踝），建议**不要频繁移动镜头**。
    4.  尽可能选择光线充足（白天室外或明亮的室内球馆）的环境拍摄，不要背光。

---

准备好视频后（例如 `my_match.mp4`），就可以使用以下命令运行测试：
```bash
python demos/pickleball_demo.py --video my_match.mp4
```
